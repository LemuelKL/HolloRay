This is directory SLELabs\Syntax which contains examples for use with
the OSBRD parser generator embedded with ART.

To run example EX1.art on Windows, type

parse EX1

Note that there is no filetype. If you type parse EX1.art you will get an error.

To run example EX1.art on Un*x, first ensure the scripts are
executable by typing chmod +x *.sh, then type

./parse EX1

Note that there is no filetype. If you type parse EX1.art you will get an error.

To clean up the directory, type clean or ./clean.sh
